This project serves as the company's digital presence, showcasing software development, web development, automation, and technology consulting services.

The website was designed with a focus on performance, responsiveness, accessibility, and lead generation for prospective clients.

## Installation

Clone the repository:

```bash
git clone https://github.com/yapisango/yapitech-website.git
```

Navigate to the project:

```bash
cd yapitech-website
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
